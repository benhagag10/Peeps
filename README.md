# Peeps

A visual network mapping tool for tracking your professional and personal connections.

## Features

- **Interactive Graph Canvas** - Drag, zoom, and pan to navigate your network
- **Add People** - Double-click canvas or use the Add button
- **Create Connections** - Drag from one person to another to link them
- **Auto-Connections** - People with the same stream or shared interests are automatically connected
- **Search** - Quickly find people by name or affiliation
- **Dark Mode** - Toggle between light and dark themes
- **Persistent Storage** - Data saved locally in your browser

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Tech Stack

- React 18 + TypeScript
- Vite
- @xyflow/react (graph visualization)
- Zustand (state management)
- Tailwind CSS

## Privacy

All data is stored locally in your browser's localStorage. No data is sent to any server.
